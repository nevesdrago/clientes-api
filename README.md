# Descrição

Projeto que usa JavaScript, HTML e CSS para criar uma lista de clientes, sendo possível adicionar e remover novos clientes. 

## Como funciona 

A aplicação, através do fetch API, faz uma requisição ao endpoint CRUDCRUD, que retorna os dados adicionados, exibindo-os na tela.

## Como usar 

<img width="564" height="412" alt="image" src="https://github.com/user-attachments/assets/cc0f5592-ee2e-4708-a78d-92ed81062a09" />

1. Preencha o campo de nome e email do cliente.

2. Clique no botão adicionar.

3. E pronto! O cliente vai ser exibido na tela.

4. Se desejar, clique no botão vermelho X para remover o cliente.

## CRUDCRUD

Como esse projeto foi feito apenas para fins educativos, usei o plano gratuito do CRUDCRUD. 
Após 24hrs, o endpoint vai parar de receber requisições.

### Como resolver

1. Entre no site do CRUDCRUD: https://crudcrud.com/.

2. Copie o link do endpoint <img width="1249" height="401" alt="image" src="https://github.com/user-attachments/assets/87efdf2f-210d-4582-a130-90d260e16f7f" />

3. No vscode, no arquivo scrips.js, substitua o link do endpoint pelo seu novo link. Lembre-se de manter o "/clientes" no final da URL.
